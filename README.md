# BLUE N WATER Website v1.21

정적 HTML/CSS/JS 기반 홈페이지입니다. GitHub Pages에 그대로 업로드 가능합니다.

## 배포
1. GitHub Repository 생성
2. 이 폴더의 모든 파일을 루트에 업로드
3. Settings > Pages > Deploy from a branch > main / root 선택
4. Custom domain에 도메인 입력

## 이미지 교체
- 메인 히어로 이미지: `assets/images/hero/water-cover.png` 교체
- 추가 히어로 이미지 권장 파일명: `assets/images/hero/hero-main.jpg`
- 기술 이미지: `assets/images/technology/`
- 프로젝트 이미지: `assets/images/projects/`
- 영상: `assets/videos/`

## 언어
- 현재 언어 전환 데이터는 `assets/js/language.js`에 포함되어 있습니다.
- 추후 `assets/lang/ko.json`, `en.json`, `ja.json` 구조로 분리 가능.
